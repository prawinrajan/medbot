# Mech pp

Content sharing application
# Features

1. Quick Splash Screen
2. click to Login & Sign up Using Google
3. Interactive UI design
4. Dark Mode
5. Readable Stories
6. 1 click to Feedback
7. App size (5.5 MB)
8. with Session Management
